# Directory Viewer

Directory Viewer is a Python application that provides a graphical user interface for browsing and managing directories and files on your system.

## Features

- Browse directories and files in a tree-like structure
- View file details including size, modification date, and type
- Open files and folders directly from the application
- Search for files and directories
- Responsive GUI built with PyQt6

## Files

The project consists of two main Python files:

1. `directory_viewer_logic.py`: Contains the core logic for directory and file operations[1].
2. `directory_viewer_gui.py`: Implements the graphical user interface using PyQt6[2].

## Requirements

- Python 3.x
- PyQt6

## Usage

Run the application by executing the `directory_viewer_gui.py` file:

```
python directory_viewer_gui.py
```
